 ## '<h2>' Review '<h2>'
Convert an image or video from dark mode to light mode using [ffmeg] [(https://www.ffmpeg.com)].

```bash
ffmeg -i input.mp4 -vf "negate,hue=h=180,eq=contrast=1.2:saturation=1.1" output.p4
```
